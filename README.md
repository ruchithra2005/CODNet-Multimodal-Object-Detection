# CODNet – Multimodal Object Detection Framework

CODNet is a multimodal object detection framework designed for vision-language understanding and accurate object localization.

## Features
- Multimodal object detection
- Vision-language integration using BLIP
- Feature extraction using VGG19
- Spatial modeling using LSTM
- Bounding box localization using YOLOv8
- Confidence-ranked detections using NMS

## Tech Stack
- Python
- PyTorch
- YOLOv8
- BLIP
- VGG19
- LSTM

## Dataset
- MS-COCO 2017 Dataset

## Results
- Achieved 96.5% AvgWA on MS-COCO 2017 benchmark dataset.
